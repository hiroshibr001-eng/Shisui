BrunoFrameOverlay V3.6 — Render Scale Native/Vulkan equilibrado

Aplicar patch-only:
1. Extraia este ZIP por cima da pasta atual do projeto.
2. NÃO apague app/src/main/jniLibs/arm64-v8a/libbrunolsfg.so.
3. Limpe: rm -rf app/build build .gradle
4. Compile no AndroidIDE.

IMPORTANTE:
V3.6 altera native/brunolsfg_native.cpp e NativeLsfgBridge.java.
Para Render Scale funcionar também no motor nativo/Vulkan, gere uma nova libbrunolsfg.so pelo GitHub Actions:

1. Suba a pasta atualizada para o GitHub.
2. Abra Actions -> Build Bruno LSFG Native Engine.
3. Aguarde ficar verde.
4. Baixe brunolsfg-arm64-v8a.zip.
5. Copie libbrunolsfg.so para app/src/main/jniLibs/arm64-v8a/libbrunolsfg.so.
6. Recompile o APK no AndroidIDE.

O que mudou:
- Render Scale agora é enviado para a .so.
- Native recebe inputWidth/inputHeight e outputWidth/outputHeight.
- Status mostra native render menor -> fullscreen.
- Economia agora é equilibrada; não reduz multiplier nem flow.
- Menos agressivo que V3.5 para não afetar geração IA.
